触屏事件

移动设备上的事件与PC上完全不一样。

下面的事件，除了ms-click外，全部使用ms-on-*来使用。

事件名说明
tap单击,消除手机上300ms延时的问题
doubletap双击
longtap长按
swipeleft向左划
swiperight向右划
swipeup向上划
swipedown向下划

如何在PC上体验？

安装chrome最新版，F12-->左下角的设定--〉Override User Agent --〉Emulate Touch Event (打勾) -->在打开调试工具的情况下刷新页面，就会看到光标变成一个大黑点了详看这里

点击穿透问题
