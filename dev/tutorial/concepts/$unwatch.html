$unwatch方法

$unwatch(type? , fn? ), 第一个参数是VM上的某个监控属性的名字, 第二个参数是$watch回调， 它共有三种用法

  • 如果两个参数都存在， 相当于将传入的回调，从该监控属性对应的数组移除此回调
  • 如果只有一个参数type， 相当于将该监控属性对应的数组置换为一个空数组
  • 如果没有参数， 相当于将该VM上的所有监控属性的回调都临时清空

一个VM的结构大致如下，比如它有两个监控属性a，b，那么它的$events对象就必须有两个同名的数组a,b；当我们$watch都某个回调时， 相当于将此回调放进对应数组；然后当监控属性发生变动时，就会依次触发里面的回调；当我们$unwatch它时，就会移除此回调或清空此数组

{ a: 1, b: 1, $id: "test", $events: { a: [], b: [] } }
